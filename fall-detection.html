Fall Detection Prototype

Goal: Detect falls in an elderly user without requiring a wearable device.

Constraints

  • User refuses wearable devices
  • Must work on carpet and hard flooring
  • Long battery life required

System Overview

  • ESP32 microcontroller
  • IMU (MPU-6050)
  • Vibration sensor
  • WiFi notifications

What I Did

  • Defined fall-detection thresholds using acceleration + vibration
  • Built and tested multiple sensor configurations
  • Evaluated false positives during normal activity

Results & Learnings

  • Reduced false positives compared to vibration-only detection
  • Identified limitations of floor-mounted sensing
